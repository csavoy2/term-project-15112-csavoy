Weekly Planner:
This app is a weekly planner app where users can input data for 4 categories: to do, events, notes, and highlights (category names and number are customizable). There is an option to see a four week view and edit/add to past and future weeks. Users can also check off and underline text.

How to run this project: To run this app, the user should run TPCode.py. The user txt files are all stored in a ‘Users’ folder within the TP folder.

Libraries/Modules: None need to be installed.

Shortcut Commands:
To underline characters (once a box is selected), press ~. To turn it off, press -.
Toggle edit mode with Down and Up
Toggle check mode with + (for on and off)
Press | (the character, not the letter) to take a snapshot.
